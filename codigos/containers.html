¿Qué es un Container en Bootstrap 5?

En Bootstrap, un container es un elemento fundamental que ayuda a organizar y centrar el contenido de una página web. Siempre es recomendable que toda estructura comience dentro de un container.

Existen tres tipos principales de containers:

  • .container: ancho adaptativo, centrado.
  • .container-fluid: ocupa el 100% del ancho.
  • .container-{breakpoint}: se vuelve fijo a partir de un tamaño específico.
