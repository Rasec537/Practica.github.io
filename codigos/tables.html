Módulo: Tablas en Bootstrap 5

En este módulo aprenderás cómo crear tablas responsivas utilizando Bootstrap 5. Veremos desde tablas básicas hasta tablas interactivas con filas expandibles, colores, alertas y botones de acción para una mejor experiencia de usuario.

Cómo construimos la tabla

Usamos las clases table, table-hover, table-striped y table-bordered. Cada fila representa un estudiante e incluye su foto, nombre, calificación y botones de acción.

Para mostrar fotos circulares usamos img-thumbnail y rounded-circle. Los botones btn-warning y btn-danger permiten editar o eliminar registros. Además, agregamos filas expandibles usando collapse para mostrar detalles adicionales.

Cómo organizar contenido con columnas

Usamos la clase row para crear filas y col-md-4 para dividir la página en tres columnas iguales. Esto permite mostrar contenido adicional de forma ordenada.
